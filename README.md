This repository contains the source code for a **Note-Taking Application**. It is designed to help users capture thoughts, organize tasks, and manage their daily notes efficiently.

Key goals of this project:
* **Simplicity:** A clean and distraction-free interface.

## âš¡ Getting Started

You can run this project locally without any complex installation.

### Prerequisites
You only need a modern web browser (Chrome, Firefox, Edge, etc.).

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/raj-yadav004/Note-App.git](https://github.com/raj-yadav004/Note-App.git)
    ```

2.  **Navigate to the project folder**
    ```bash
    cd Note-App
    ```

3.  **Run the project**
    * Simply double-click `index.html` to open it in your browser.
    * **OR** if using VS Code, use the "Live Server" extension.
